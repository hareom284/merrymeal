{% comment %} Service worker registration + Android install hint. The script no-ops on browsers without ``navigator.serviceWorker`` so legacy desktop browsers don't see a console error. ``catch`` swallows registration failures (e.g. localhost with self-signed certs over HTTPS) — the site still works without the SW, the user just doesn't get offline support or the install prompt. Android Chrome fires ``beforeinstallprompt`` once the PWA install criteria are met (manifest + SW + icon + start_url reachable). We stash the event and expose an "Install app" button via a small Alpine snippet, because Chrome no longer shows the URL-bar install icon automatically on mobile. {% endcomment %}
