## User story

> **As an** admin
> **I want** to click one button and download a monthly board pack with the
> same numbers in both CSV and PDF
> **so that** I do not spend Sunday afternoon assembling numbers from five queues.

## Acceptance criteria

- [ ] Route `/admin/reports/board/?month=YYYY-MM`. `month` defaults to
      the previous calendar month if omitted.
- [ ] Output format selected by `?format=`:
  - `csv` (default) — a `.zip` archive of CSV "sheets".
  - `pdf` — one-page summary PDF.
- [ ] **Sheets / sections** (both formats):
  1. **Meals delivered** — rows: kitchen × day (date), value: delivered
     count.
  2. **Donations per campaign** — rows: campaign, value: total cents +
     count.
  3. **Volunteer hours / route count** — rows: volunteer, value: route
     count for the month. (Hours = route count × placeholder; flagged
     for Epic 07 once route start/end timestamps land.)
  4. **Failed deliveries** — rows: kitchen + reason (or date), value:
     count.
  5. **Food-safety check pass-rate** — pass count, fail count, % pass.
- [ ] **Rate limit:** an admin may generate a report at most 5 times per
      minute. 6th call → 429.
- [ ] **Audit log entry** every time a report is generated:
      `actor`, `report_type`, `month`, `format`, `generated_at`.
- [ ] PDF renders in Docker (CI smoke test asserts the PDF contains the
      section titles).
- [ ] Time zone: Australia/Melbourne for "month" boundaries.

## Tech / library notes

- **WeasyPrint Debian deps.** WeasyPrint needs:
  ```
  libpango-1.0-0 libpangoft2-1.0-0 libharfbuzz0b
  libcairo2 libgdk-pixbuf-2.0-0 shared-mime-info
  fonts-liberation
  ```
  Add to the runtime stage of `Dockerfile` (NOT the css-builder stage).
- **Rate limiting.** Use `django-ratelimit`:
  ```python
  from django_ratelimit.decorators import ratelimit
  @ratelimit(key="user", rate="5/m", block=True)
  ```
  Alternative (no new dep): a small cache-based counter with
  `cache.incr` per (`user_id`, minute_bucket). Pick whichever ships
  faster; the test below works for both.
- **Audit log entry.** Use `django-auditlog`'s `LogEntry.objects.log_create`
  with a synthetic `Report` object, OR write a small bespoke
  `BoardReportAudit` model. Recommended: bespoke model — it keeps the
  audit-row schema obvious for Story 6.6.
- **CSV "sheets".** Bundle as a `.zip` of five CSVs in memory:
  `01-meals-delivered.csv`, `02-donations.csv`, etc. Stream with
  `HttpResponse(..., content_type="application/zip")`.
- **PDF rendering.** Render an HTML template with the same data
  passed to the CSV builder, then `weasyprint.HTML(string=html).write_pdf()`.
  The HTML uses `@page { size: A4 landscape; margin: 12mm; }`.
- **Month boundaries.** `month=YYYY-MM` → first day of that month at
  00:00 Mel, exclusive end of next month.
- **Read-only.** This story does NOT write business data; only writes
  the audit row.

```python
from django.db import models


class BoardReportAudit(models.Model):
    actor_id = models.BigIntegerField()
    report_type = models.CharField(max_length=40, default="board")
    month = models.CharField(max_length=7)  # YYYY-MM
    format = models.CharField(max_length=8)  # csv | pdf
    generated_at = models.DateTimeField(auto_now_add=True)

    class Meta:
        app_label = "dashboards"
        db_table = "board_report_audit"
        indexes = [models.Index(fields=["actor_id", "generated_at"])]

    def __str__(self) -> str:
        return f"{self.report_type} {self.month} by {self.actor_id}"
```

### Task 3: Data collection service (GREEN — part 2)

- [ ] **Step 3.1 — Create `apps/dashboards/services/board_report.py`.**

```python
from __future__ import annotations

from datetime import date
from datetime import datetime, timedelta

from django.db.models import Count, Sum
from django.utils import timezone


def _month_bounds(month: str) -> tuple[date, date]:
    """Return (first_day, last_day_inclusive) for YYYY-MM."""
    y, m = (int(x) for x in month.split("-"))
    first = date(y, m, 1)
    if m == 12:
        next_first = date(y + 1, 1, 1)
    else:
        next_first = date(y, m + 1, 1)
    return first, next_first - timedelta(days=1)


def collect(month: str) -> dict:
    start, end = _month_bounds(month)

    return {
        "month": month,
        "meals_delivered": _meals_delivered(start, end),
        "donations_per_campaign": _donations_per_campaign(start, end),
        "volunteer_routes": _volunteer_routes(start, end),
        "failed_deliveries": _failed_deliveries(start, end),
        "food_safety_pass_rate": _food_safety_pass_rate(start, end),
    }


def _meals_delivered(start, end) -> list[dict]:
    from apps.deliveries.models import Delivery
    qs = (
        Delivery.objects.filter(
            status="delivered",
            delivered_time__date__gte=start,
            delivered_time__date__lte=end,
        )
        .values("meal_plan__kitchen__name", "delivered_time__date")
        .annotate(count=Count("id"))
        .order_by("meal_plan__kitchen__name", "delivered_time__date")
    )
    return list(qs)


def _donations_per_campaign(start, end) -> list[dict]:
    from apps.fundraising.models import Donation
    qs = (
        Donation.objects.filter(
            status="completed",
            created_at__date__gte=start,
            created_at__date__lte=end,
        )
        .values("campaign__name")
        .annotate(total_cents=Sum("amount_cents"), count=Count("id"))
        .order_by("campaign__name")
    )
    return list(qs)


def _volunteer_routes(start, end) -> list[dict]:
    from apps.deliveries.models import Route
    qs = (
        Route.objects.filter(
            route_date__gte=start, route_date__lte=end,
            status__in=["completed", "in_progress"],
        )
        .values("volunteer__full_name")
        .annotate(route_count=Count("id"))
        .order_by("volunteer__full_name")
    )
    return list(qs)


def _failed_deliveries(start, end) -> list[dict]:
    from apps.deliveries.models import Delivery
    qs = (
        Delivery.objects.filter(
            status="failed",
            scheduled_date__gte=start, scheduled_date__lte=end,
        )
        .values("meal_plan__kitchen__name", "scheduled_date")
        .annotate(count=Count("id"))
        .order_by("meal_plan__kitchen__name", "scheduled_date")
    )
    return list(qs)


def _food_safety_pass_rate(start, end) -> dict:
    from apps.kitchens.models import FoodSafetyCheck
    qs = FoodSafetyCheck.objects.filter(
        checked_at__date__gte=start, checked_at__date__lte=end,
    )
    total = qs.count()
    if total == 0:
        return {"pass": 0, "fail": 0, "pct": None}
    passed = qs.filter(result="pass").count()
    return {
        "pass": passed,
        "fail": total - passed,
        "pct": round(passed / total * 100, 1),
    }
```

```python
import csv
import io
import zipfile


def build_zip(data: dict) -> bytes:
    """Pack the board-report dict into 5 CSVs inside a single zip."""
    sheets = {
        "01-meals-delivered.csv": (
            ["kitchen", "date", "count"],
            [(r["meal_plan__kitchen__name"],
              r["delivered_time__date"].isoformat(), r["count"])
             for r in data["meals_delivered"]],
        ),
        "02-donations-per-campaign.csv": (
            ["campaign", "total_cents", "count"],
            [(r["campaign__name"], r["total_cents"], r["count"])
             for r in data["donations_per_campaign"]],
        ),
        "03-volunteer-routes.csv": (
            ["volunteer", "route_count"],
            [(r["volunteer__full_name"], r["route_count"])
             for r in data["volunteer_routes"]],
        ),
        "04-failed-deliveries.csv": (
            ["kitchen", "date", "count"],
            [(r["meal_plan__kitchen__name"],
              r["scheduled_date"].isoformat(), r["count"])
             for r in data["failed_deliveries"]],
        ),
        "05-food-safety.csv": (
            ["pass", "fail", "pct"],
            [(data["food_safety_pass_rate"]["pass"],
              data["food_safety_pass_rate"]["fail"],
              data["food_safety_pass_rate"]["pct"])],
        ),
    }
    buf = io.BytesIO()
    with zipfile.ZipFile(buf, "w", zipfile.ZIP_DEFLATED) as zf:
        for filename, (header, rows) in sheets.items():
            csv_buf = io.StringIO()
            w = csv.writer(csv_buf)
            w.writerow(header)
            w.writerows(rows)
            zf.writestr(filename, csv_buf.getvalue())
    return buf.getvalue()
```

```python
from datetime import date

from django.http import HttpResponse
from django_ratelimit.decorators import ratelimit

from apps.core.decorators import role_required
from apps.dashboards.models import BoardReportAudit
from apps.dashboards.services import board_report
from apps.dashboards.services.csv_zip import build_zip
from apps.dashboards.services.pdf import render_pdf


def _default_month() -> str:
    today = date.today()
    if today.month == 1:
        prev = date(today.year - 1, 12, 1)
    else:
        prev = date(today.year, today.month - 1, 1)
    return prev.strftime("%Y-%m")


@ratelimit(key="user", rate="5/m", block=True)
@role_required("admin")
def board_report_view(request):
    month = request.GET.get("month") or _default_month()
    fmt = request.GET.get("format", "csv")
    data = board_report.collect(month=month)

    BoardReportAudit.objects.create(
        actor_id=request.user.id, month=month, format=fmt,
    )

    if fmt == "pdf":
        pdf_bytes = render_pdf(
            "dashboards/admin/board_report.html", {"data": data}
        )
        resp = HttpResponse(pdf_bytes, content_type="application/pdf")
        resp["Content-Disposition"] = f'attachment; filename="board-{month}.pdf"'
        return resp

    zip_bytes = build_zip(data)
    resp = HttpResponse(zip_bytes, content_type="application/zip")
    resp["Content-Disposition"] = f'attachment; filename="board-{month}.zip"'
    return resp
```

> `django-ratelimit` returns a 403 by default on block; configure
> `RATELIMIT_VIEW = "apps.dashboards.views.board_report.too_many_requests"`
> in settings that returns a 429. Alternatively, wrap the view with a
> small handler that translates the `Ratelimited` exception.

## Pitfalls reviewer will check

- **WeasyPrint missing libs in Docker.** Symptom: `OSError: cannot load
  library 'libpango-1.0-0'`. Fix: add the apt deps in the runtime
  stage; rebuild the image.
- **`fonts-liberation` missing.** PDF renders but every word is a box
  glyph.
- **Rate limit applied before auth.** Decorator order matters: place
  `@ratelimit` above `@role_required`; otherwise anonymous spammers
  consume the admin's quota.
- **Audit row never written.** Rate-limited rejections should NOT create
  an audit row — verify the order of operations.
- **Month parsed naively.** `"2026-5"` (no zero pad) should 400; do not
  silently accept.
- **PDF in test is binary.** Asserting on `b"Meals delivered"` may fail
  when PDF compresses text streams. Either disable WeasyPrint stream
  compression in tests or assert magic bytes (`%PDF-`).
- **Cents shown as cents in PDF.** Without a divide-by-100, $2500
  donations show as `2500` instead of `$25.00`.
