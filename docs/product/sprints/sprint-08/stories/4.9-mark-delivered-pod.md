## User story

> **As** Sarah, a volunteer on a bike at Margaret's gate
> **I want** to tap "Mark delivered", let the camera capture the meal on the
> doorstep, and see the screen advance to the next stop
> **so that** I have proof of drop-off and Margaret's daughter sees a green
> tick within seconds — even if my signal is patchy.

## Acceptance criteria

- [ ] New HTMX endpoint `POST /volunteer/delivery/<id>/mark-delivered/`.
- [ ] Volunteer can only call this for a `Delivery` where
      `delivery.volunteer_id == request.user.id`; anything else → 404 (do
      not leak existence with 403).
- [ ] **Camera capture** — the CTA from Story 4.8 opens
      `<input type="file" accept="image/*" capture="environment">`.
      Submitting the form sends the photo as `multipart/form-data`.
- [ ] **Upload** — file uploaded to S3 via `django-storages`; the resulting
      public/cdn URL is written to `Delivery.photo`.
- [ ] **HEIC → JPEG** — if the uploaded file is HEIC/HEIF (iOS Safari
      default), the server converts to JPEG via Pillow + `pyheif` before
      uploading to S3. Only the JPEG URL is stored.
- [ ] **Status + audit** — `Delivery.status = 'delivered'`,
      `delivered_time = timezone.now()`. An `auditlog` row is written by the
      registered LogEntry config.
- [ ] **Geolocation** — the form includes hidden `latitude` / `longitude`
      inputs populated client-side from `navigator.geolocation`. If denied
      or unavailable, the inputs are left blank and the server tolerates
      null (no error, no warning toast).
- [ ] **Auto-advance** — successful HTMX response swaps the current stop
      card for the next pending one; the sticky CTA re-renders for the
      new "current" stop.
- [ ] **Offline queue** — if the network request fails (offline, 5xx,
      timeout > 8 s), the volunteer-offline.js script:
  - Pushes `{delivery_id, photo_blob_dataurl, lat, lng, queued_at}` onto a
    `localStorage` JSON list under key `merrymeal_pod_queue`.
  - Renders a `queued — will retry` badge on that stop card (orange).
  - On `window.online` event, replays each queued item in order; on success,
    removes it from the queue and updates the badge to a green tick.
- [ ] **Server-side queue logging** — every successful mark-delivered also
      writes a structured log line `pod.delivered delivery=<id> volunteer=<id>`
      so we can reconcile server vs client queue (mitigates the Sprint 08
      risk "offline queue bugs eat a delivery").

## Tech / library notes

- **`django-storages` in prod only.** Dev / test must keep the default
  `FileSystemStorage` writing to `media/`. Otherwise CI needs AWS creds.
- **S3 bucket** — `merrymeal-pod-{env}`; lifecycle rule: delete objects
  after 180 days (Sprint 08 DoD).
- **`pyheif`** wraps `libheif`. On macOS dev, `brew install libheif`.
  On the Docker image (Debian-slim from Story 0.9), apt-install
  `libheif-dev`.
- **Timeout** — wrap the upload in a `socket.setdefaulttimeout(8)` window
  so a flat tunnel triggers the offline queue instead of hanging the UI.
- **HTMX response** — return the rendered next stop card as a partial; use
  `hx-swap="outerHTML"` and `hx-target="closest [data-testid=stop-card]"`.
  For "no more stops", swap in an "All done — thank you" card.
- **Audit log** — `django-auditlog` is already registered for `Delivery`
  in earlier sprint; we do **not** call `LogEntry.objects.create` manually.
- **Geolocation API** — call `navigator.geolocation.getCurrentPosition`
  with `{ timeout: 4000, maximumAge: 30000 }`. On failure, leave hidden
  inputs blank — do **not** alert.
- **CSRF for HTMX** — already wired via `htmx.config.csrfToken` in `base.html`.

```python
"""Service: mark a Delivery as delivered with POD photo + geo."""
from django.db import transaction
from django.utils import timezone
from apps.delivery.models import Delivery


@transaction.atomic
def mark_delivered(delivery: Delivery, *, photo_url: str, lat, lng) -> Delivery:
    """Transition Delivery to 'delivered'. Idempotent on re-call."""
    locked = Delivery.objects.select_for_update().get(pk=delivery.pk)
    if locked.status == "delivered":
        return locked  # idempotent — keep original time/photo

    locked.status = "delivered"
    locked.delivered_time = timezone.now()
    locked.photo = photo_url
    locked.latitude = lat
    locked.longitude = lng
    locked.save(update_fields=["status", "delivered_time", "photo",
                               "latitude", "longitude", "updated_at"])
    return locked
```

```python
"""HEIC → JPEG conversion + S3 upload helper."""
import io
from django.core.files.storage import default_storage
from django.core.files.base import ContentFile
from PIL import Image

try:
    import pyheif
except ImportError:  # libheif not present (dev box without brew install)
    pyheif = None


HEIC_TYPES = {"image/heic", "image/heif"}


def convert_to_jpeg(fh, content_type: str):
    """Return (BytesIO, 'image/jpeg'). HEIC → JPEG; JPEG/PNG → re-encoded JPEG."""
    if content_type in HEIC_TYPES:
        if pyheif is None:
            raise RuntimeError("libheif missing — install pyheif/libheif")
        heif = pyheif.read(fh.read())
        img = Image.frombytes(heif.mode, heif.size, heif.data,
                              "raw", heif.mode, heif.stride)
    else:
        img = Image.open(fh).convert("RGB")

    out = io.BytesIO()
    img.save(out, format="JPEG", quality=85, optimize=True)
    out.seek(0)
    return out, "image/jpeg"


def upload_pod_photo(uploaded_file, *, delivery_id: int) -> str:
    """Convert + upload to S3 (prod) or media/ (dev). Returns public URL."""
    buf, _ = convert_to_jpeg(uploaded_file, uploaded_file.content_type)
    name = f"pod/{delivery_id}/{uploaded_file.name.rsplit('.', 1)[0]}.jpg"
    path = default_storage.save(name, ContentFile(buf.read()))
    return default_storage.url(path)
```

```python
from django.http import Http404, HttpResponse
from django.shortcuts import get_object_or_404, render
from django.views.decorators.http import require_POST
from apps.delivery.models import Delivery
from apps.delivery.forms.mark_delivered import MarkDeliveredForm
from apps.delivery.services.mark_delivered import mark_delivered
from apps.delivery.services.photo import upload_pod_photo

@login_required
@role_required("volunteer")
@require_POST
def mark_delivered_view(request, pk: int):
    delivery = get_object_or_404(Delivery, pk=pk, volunteer=request.user)
    form = MarkDeliveredForm(request.POST, request.FILES)

        return HttpResponse(status=400)

        delivery,
        photo_url=url,
        lat=form.cleaned_data.get("latitude"),
        lng=form.cleaned_data.get("longitude"),
    )

    context = get_today_route(request.user)
    return render(request, "delivery/volunteer/_route_fragment.html", context)
```

```javascript
/* Story 4.9 — offline POD queue.
   Interim before Story 7.1 (proper PWA + IndexedDB). */
(function () {
  const KEY = "merrymeal_pod_queue";
  const load = () => JSON.parse(localStorage.getItem(KEY) || "[]");
  const save = (q) => localStorage.setItem(KEY, JSON.stringify(q));

  // Populate geo hidden inputs on page load.
  function populateGeo() {
    if (!navigator.geolocation) return;
    navigator.geolocation.getCurrentPosition(
      (pos) => {
        document.querySelectorAll("[data-geo=lat]").forEach(
          (el) => (el.value = pos.coords.latitude.toFixed(7))
        );
        document.querySelectorAll("[data-geo=lng]").forEach(
          (el) => (el.value = pos.coords.longitude.toFixed(7))
        );
      },
      () => {}, // denied — leave blank
      { timeout: 4000, maximumAge: 30000 }
    );
  }

  function enqueue(form, file) {
    const reader = new FileReader();
    reader.onload = () => {
      const queue = load();
      queue.push({
        delivery_id: form.dataset.deliveryId,
        url: form.action,
        photo: reader.result,
        lat: form.querySelector("[data-geo=lat]")?.value || "",
        lng: form.querySelector("[data-geo=lng]")?.value || "",
        queued_at: Date.now(),
      });
      save(queue);
      markBadge(form, "queued");
    };
    reader.readAsDataURL(file);
  }

  function markBadge(form, state) {
    const card = form.closest("[data-testid=stop-card]");
    if (!card) return;
    let badge = card.querySelector("[data-testid=queue-badge]");
    if (!badge) {
      badge = document.createElement("span");
      badge.dataset.testid = "queue-badge";
      badge.className =
        "inline-block px-2 py-1 text-[14px] rounded bg-brand-orange text-white";
      card.prepend(badge);
    }
    badge.textContent =
      state === "queued" ? "queued — will retry" : "✓ synced";
    badge.className = badge.className.replace(
      state === "queued" ? "bg-brand-green" : "bg-brand-orange",
      state === "queued" ? "bg-brand-orange" : "bg-brand-green"
    );
  }

  async function flush() {
    const queue = load();
    const remaining = [];
    for (const item of queue) {
      try {
        const fd = new FormData();
        const blob = await (await fetch(item.photo)).blob();
        fd.append("photo", blob, "pod.jpg");
        fd.append("latitude", item.lat);
        fd.append("longitude", item.lng);
        fd.append("csrfmiddlewaretoken",
          document.querySelector("[name=csrfmiddlewaretoken]").value);
        const r = await fetch(item.url, { method: "POST", body: fd });
        if (!r.ok) throw new Error("server " + r.status);
      } catch (e) {
        remaining.push(item);
      }
    }
    save(remaining);
    if (remaining.length === 0) {
      document
        .querySelectorAll("[data-testid=queue-badge]")
        .forEach((b) => (b.textContent = "✓ synced"));
    }
  }

  document.addEventListener("htmx:responseError", (e) => {
    const form = e.detail.elt.closest("[data-pod-form=1]");
    if (form) {
      const file = form.querySelector("input[type=file]").files[0];
      if (file) enqueue(form, file);
    }
  });
  window.addEventListener("online", flush);
  document.addEventListener("DOMContentLoaded", () => {
    populateGeo();
    if (navigator.onLine) flush();
  });
})();
```

## Pitfalls reviewer will check

- **HEIC defaults on iOS** — uploads will be `image/heic`; without
  `pyheif`, Pillow throws `UnidentifiedImageError`. Confirm conversion
  before merging.
- **`@require_POST` skipped** — a missing decorator lets a GET demo-link
  silently flip status. Required.
- **`select_for_update` inside transaction** — service must hold the row
  to make the idempotency check race-safe.
- **`enctype="multipart/form-data"`** missing on the form → file silently
  not uploaded.
- **HTMX + multipart** — `hx-encoding="multipart/form-data"` is mandatory
  (HTMX defaults to urlencoded).
- **localStorage size** — a single 5 MB photo fills the quota fast. Cap
  enqueued items at 20 and drop oldest with a console.warn — Story 7.1
  will move this to IndexedDB.
- **Audit log spam** — repeated mark-delivered on an already-delivered
  row must short-circuit before `save()` to avoid noisy audit entries.
- **`AWS_DEFAULT_ACL = None`** — required since AWS deprecated ACLs in
  new buckets (2023). Otherwise uploads 403.
- **Geolocation prompt in iframe** — Playwright must grant permission:
  `context: { permissions: ['geolocation'] }` if you want lat/lng tests.
